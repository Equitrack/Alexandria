Python es un lenguaje de programación interpretado, versátil, amigable y popular, con el que se pueden hacer muchas automatizaciones. 

Características:

- Tipado dinámico
- Gestión automática de memoria
- Multiplataforma
- Multiparadigma
- Gran ecosistema de bibliotecas
- Comunidad activa
